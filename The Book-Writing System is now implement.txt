The Book-Writing System is now implemented. Here's how to use it:

Quick Start
1. Create a Volume Config File
Create config_vol1.json in your project root:

{
  "author": "Your Name",
  "bookTitle": "Your Book Title",
  "bookSubtitle": "A Subtitle",
  "volume": "Volume 1: Foundation",
  "edition": "First Edition, 2026",
  "publisher": "Your Publisher",
  "cefr": "A1-A2",
  "year": "2026",
  "chaptersDir": "./chapters",
  "outputDir": "./output",
  "chapterManifest": [
    { "file": "v1_ch1_intro.js", "num": "1", "title": "Introduction", "titleEn": "Introduction", "type": "core" }
  ]
}
2. Create Chapter Files
Create chapters in the ./chapters directory using the template format:

if (typeof window !== 'undefined') {
  window.LEKHAK_REPOSITORY = window.LEKHAK_REPOSITORY || {};
}

window.LEKHAK_REPOSITORY["v1_c1"] = `
# Chapter Title
## Chapter Title in English

Your content here...
`;
3. Compile
Run from project root:

node compile_vol1.js
Or compile all volumes:

node -e "const {compileAllVolumes} = require('./src/multi-volume-manager'); console.log(JSON.stringify(compileAllVolumes('.'), null, 2));"
4. Output
The compiler generates:

manuscript_v1.md
 - Raw markdown
manuscript_v1.html
 - Print-ready HTML with CSS
Open the HTML file in a browser and click "Print / Save PDF" to generate your book.
